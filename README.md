**住宅代理（Residential Proxies）**是他们家的主打产品。这种代理用的是真实用户的IP地址，覆盖全球195个国家和地区，IP池有3200多万个地址。因为是真实住宅网络，所以在做数据采集、市场调研这些事情的时候，被封的概率会低很多。住宅代理支持IP轮换和粘性会话，还支持SOCKS5协议，用起来比较方便。

**数据中心代理（Datacenter Proxies）**就是纯粹的服务器IP。速度快，价格便宜，但因为不是真实用户IP，所以可信度相对低一些。如果你的目标网站对IP类型不那么敏感，或者你只是需要快速访问一些基础内容，数据中心代理性价比最高。

**移动代理（Mobile Proxies）**用的是真实移动设备的4G/5G网络IP。这种代理的特点是可信度极高，因为很多网站对移动端用户的检测会宽松一些。不过移动代理价格也是四种里最贵的，一般用于需要高度匿名性或者专门针对移动端的场景。
